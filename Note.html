Important Notes

Harap baca informasi berikut sebelum menggunakan tool agar proses preview dan pengunduhan sticker berjalan dengan lancar.

Format File yang Didukung

Pastikan file .zip yang diunggah berisi sticker dengan format .webp, .webm, atau .tgs. File ZIP tanpa format yang didukung tidak dapat diproses oleh sistem.

Jangan Mengganggu Proses Preview

Harap tunggu hingga seluruh preview sticker selesai dimuat sebelum mengunggah file ZIP lain atau menekan tombol Delete All. Melakukan tindakan tersebut saat proses masih berlangsung dapat menyebabkan preview sticker tercampur, rusak, atau gagal ditampilkan dengan benar.
